## 使用原则

1. **界面显示文案**：必须使用中文
2. **技术字段名**：保持英文（如 API 字段名、数据库字段名）
3. **代码注释**：使用中文
4. **文档**：使用中文
5. **技术产品名称**：保留英文（如 Dify、Provider、API）
6. **专有名词**：保留英文（如 HTTP、JSON、SQL）
