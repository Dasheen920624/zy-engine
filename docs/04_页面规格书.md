### API

| 用途 | API |
|---|---|
| 加载草稿 | `GET /api/pathways/{code}` |
| 自动保存 | `PUT /api/pathways/{code}/draft`（每 30s 触发，去抖） |
| 校验完整性 | `POST /api/pathways/{code}/validate` |
| 提交审核 | `POST /api/pathways/{code}/submit-review` |
| 获取可绑定规则 | `GET /api/rules?status=published&dept={}` |
| 获取图谱可查询 | `GET /api/graph/queryable?dept={}` |
| 获取 Dify 模板 | `GET /api/dify/templates?dept={}` |

### 状态机

```
[初始加载] → [编辑中] →（自动保存）→ [已保存]
                      →（提交审核）→ [审核中]
                      →（手动保存）→ [保存中] → [已保存]
                      →（校验失败）→ [校验错误] → 修正后回到编辑
                      →（离开页面 dirty）→ [未保存提示]
```

### 空错态

- 加载失败：全屏 `<ErrorState>`
- 节点未选中：右侧面板显示"请在画布中选择节点"
- 节点孤立：画布上节点边框标红

### 权限

仅 R05 路径专家可访问；R06 主任只能进只读模式（`?mode=view`）

### 性能

- 自动保存去抖 30s + 30 节点编辑 lag-free（X6 性能基线）
- 校验 P95 < 1s

### 验收 DoD

1. ☐ 三栏可拖拽调整宽度（Allotment）
2. ☐ 节点拖拽流畅 60fps（30 节点内）
3. ☐ 自动保存生效，离开有 prompt
4. ☐ 节点属性变更触发画布刷新
5. ☐ 节点绑定规则/图谱/Dify 必须显示来源审核状态
6. ☐ 提交审核前必须通过完整性校验
