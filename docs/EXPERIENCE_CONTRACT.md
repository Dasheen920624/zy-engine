# 共享体验与组件契约（EXPERIENCE_CONTRACT）

> 定位：**页面卡**读本契约获取可复用的页面/组件模式；引擎/API 卡不必读。
> 全局硬约束（一页一目标、≤3 默认筛选、≤1 主按钮、AI 明显标识、菜单 IA 锁、设计 token）在 [核心 CONSTITUTION](CONSTITUTION.md) §产品体验 与 §设计token；本契约**不重复核心原文**，只定可复用模式。
> 来源：由 `MEDKERNEL_PRODUCT_EXPERIENCE_RULES.md` 转化（页面/组件级模式入此，全局规则上提核心）。

## 页面类型与固定结构

| 类型 | 用途 | 固定结构 |
|---|---|---|
| 工作台 | 看状态、风险、价值、待办 | 摘要卡 + 待办 + 风险 + 最近变化 |
| 列表页 | 找对象、筛选、批量、进详情 | 筛选 + 表格 + 服务端分页 + 详情抽屉 |
| 配置页 | 导入、校验、影响、审核、发布 | 7 步流 |
| 审核台 | 判断候选是否可用 | 队列 + 左右对照 + 影响 + 审核结论 |
| 详情页 | 看一个对象全貌 | 摘要 + 状态 + 来源 + 版本 + 审计 |
| 嵌入面板 | 院内系统里给建议 | 小卡片 + 动作 + 解释抽屉 + 反馈 |
| 驾驶舱 | 看趋势和责任 | 指标 + 风险 + 下钻 + 行动 |
| 专家工具 | 图谱、DSL、JSON、trace | 默认折叠，需权限进入 |

## 页面交互固定规则

| 项 | 固定规则 |
|---|---|
| 主按钮 | 每页最多 1 个；其它动作用次按钮/菜单/抽屉 |
| 默认筛选 | ≤3；更多进"高级筛选" |
| 表格列 | 默认 ≤8 列；技术字段/长文本/审计字段进详情 |
| 详情 | 长列表用右侧详情抽屉；打开详情不刷新列表 |
| 危险动作 | 删除/撤回/回滚/全量发布必须二次确认 + 展示影响范围 |
| 草稿保护 | 长表单/路径编辑/规则编辑自动保存草稿 |
| 可撤销 | 批量/发布/替换/同步提供回滚点或撤销解释 |

## 六态组件契约

每个页面必须齐全六态：**加载 / 空 / 错误 / 无权限 / 部分成功 / 正常**。

| 状态 | 契约 |
|---|---|
| 加载 | 骨架屏/进度反馈，内网慢场景必给 |
| 空 | 说明为什么空 + 下一步做什么 + 是否需要权限 |
| 错误 | 中文原因 + 重试入口 + traceId 复制按钮 |
| 无权限 | 不暴露敏感数据 + 说明权限范围 + 申请入口 |
| 部分成功 | 成功数 + 失败数 + 失败明细 + 可重试 |
| 正常 | 默认角色视图，技术对象折叠到专家模式 |

## 大规模列表与性能门槛

| 场景 | 固定规则 |
|---|---|
| 普通列表 | 默认每页 20（可 50/100）；服务端分页；默认排序清晰 |
| 海量列表 | 游标分页；总数允许异步估算 |
| 审核队列 | 默认只看待我处理/高风险/新版差异；重复与旧版进已拦截视图 |
| 搜索 | 关键词/来源/知识身份/版本/组织/风险/状态/时间范围 |
| 保存视图 | 复杂筛选可存为个人视图或角色视图 |
| 批量处理 | 选本页/选筛选结果/异步任务；**高风险不可批量自动通过** |
| 导出 | 大范围必须异步，记录筛选条件/操作者/时间/审批 |
| 历史版本 | 默认当前版/上一版/待替换版；完整历史时间线分页 |

| 性能门槛 | 值 |
|---|---|
| 首屏加载 | P95 ≤ 1s（内网极端场景给骨架屏 + 进度） |
| 筛选响应 | P95 ≤ 2s |
| 详情打开 | 不触发列表重载 |
| 批量提交 | 立即返回任务 ID 与预计完成方式 |
| 导出 | 不阻塞页面；完成后通知并可追溯 |

## 临床嵌入与低打扰

| 项 | 固定规则 |
|---|---|
| 默认形态 | 在医生工作站/EMR/护理站嵌入轻量卡片或抽屉 |
| 打扰级别 | 信息提示不打断 / 重要风险弱打断 / **红线风险才强打断** |
| 动作 | 默认只有：采纳、不采纳并说明、稍后处理、查看依据 |
| 降级 | 引擎不可用时显示"智能建议暂不可用"，**不得阻断院内系统主流程** |
| 反馈 | 关闭/忽略/采纳/不采纳/稍后处理全部回传并审计 |
| 最小数据 | 嵌入启动只传当前场景必要字段 |
| 可读性 | 适配小屏、老旧浏览器、医生站低分辨率窗口 |

## 驾驶舱契约

| 规则 | 标准 |
|---|---|
| 指标有动作 | 每个指标必须能下钻到对象/责任人/科室/待办 |
| 不做装饰图 | 禁止只展示炫酷但不可行动的图表 |
| 先异常后全量 | 默认展示风险/异常/逾期/变化/趋势 |
| 价值指标 | 采纳率/误报率/漏报回溯/路径完成率/整改闭环率/医保违规减少 可配置 |
| 证据导出 | 管理视图能导出阶段报告或证据快照 |

---

## 表单/校验/发布契约

| 项 | 固定规则 |
|---|---|
| 表单分段 | >7 字段必须分组；>15 字段必须步骤化或折叠 |
| 默认值 | 能从组织/角色/包版本/历史配置推断的字段必须给默认值 |
| 实时校验 | 字段级错误就地提示；发布级错误进校验结果页 |
| 影响分析 | 发布/替换/撤回/回滚前必须显示影响组织/科室/患者/规则/路径 |
| 审核意见 | 高风险审核必须填意见；低风险可用结构化原因 |
| 发布完成 | 展示发布范围/同步状态/回滚点/证据入口 |

## 文案与视觉

| 项 | 固定规则 |
|---|---|
| 语言 | 客户可见默认中文；英文缩写仅保留 ICD/DRG/DIP/MPI/FHIR/OIDC 等行业标准 |
| 语气 | 清楚、克制、专业；不营销化/夸张化/拟人化 |
| 按钮 | 动词 + 对象，如"提交审核""开始灰度""导出证据" |
| 风险文案 | 说明风险事实 + 影响范围 + 建议动作 |
| 颜色 | 只表达状态，不做装饰；危险色只用于真实危险（值见核心 §设计token）|
| 字号 | 正文 14px；老年医生模式 ≥16pt |
| 可访问性 | 对比度、键盘可达、焦点态、表格可读性必须验收 |
| 主题 | 5 主题模式响应：default / elder / dark / eye / system |

## 与核心的边界

以下为**全局硬约束**，住在核心、不在本契约重复，页面卡通过"适用不变量"点编号引用：

- 一页一目标 / ≤1 主按钮 / ≤3 默认筛选 → 核心 §产品体验（硬约束 #6）
- AI 内容明显标识、医师确认、低打扰红线 → 核心 §临床医疗安全红线（#9/#10）
- 菜单 IA 锁（5+1 一级 / 27+5 二级）→ 核心 §菜单 IA 锁（#2/#3）
- 设计 token（色/字号/圆角/字体）→ 核心 §设计token（#5）；本契约只引用变量名，禁硬编码
- 六态/分页/低打扰/可信解释/驾驶舱下钻 → 核心 §13 铁律 + 通用拒绝项

本契约只承载"页面类型/组件/交互/性能"的**可复用模式**；任何与核心冲突处以核心为准。
