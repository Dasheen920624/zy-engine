# MedKernel License 与用量报告

> 版本：1.0 · 2026-05-24
> 任务：GA-COMM-01

## 1. License 模型

### 1.1 License 类型

| 类型 | 代码 | 说明 |
|------|------|------|
| 永久授权 | PERPETUAL | 一次买断，永久使用 |
| 年度订阅 | ANNUAL | 按年付费，到期需续费 |
| 试用版 | TRIAL | 90 天免费试用 |

### 1.2 License 等级

| 等级 | 代码 | 最大用户数 | 最大院区数 | 功能范围 |
|------|------|-----------|-----------|----------|
| 标准版 | STANDARD | 10 | 1 | 路径引擎+规则引擎+质控 |
| 专业版 | PROFESSIONAL | 50 | 3 | 标准+知识管理+CDSS |
| 旗舰版 | FLAGSHIP | 无限 | 无限 | 全功能 |

## 2. API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/commercial/license` | GET | 获取当前 License 信息 |
| `/api/commercial/license` | POST | 更新 License |
| `/api/commercial/usage` | GET | 获取用量报告 |
| `/api/commercial/feature/{code}` | GET | 检查功能是否可用 |

## 3. 到期提醒

| 剩余天数 | 提醒级别 | 日志级别 |
|----------|----------|----------|
| > 30 天 | 无 | - |
| 30 天 | 提醒续费 | INFO |
| 7 天 | 尽快续费 | WARN |
| 1 天 | 明天到期 | ERROR |
| 已过期 | 已过期 | ERROR |

## 4. 用量统计

用量报告包含：
- 活跃用户数 / 最大用户数
- API 调用次数（按接口统计）
- 功能使用次数（按功能统计）
- License 到期天数

每月初自动重置统计计数器。
