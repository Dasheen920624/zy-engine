## 1. 文档定位

按 [基础底座与引擎服务能力总览](../../MEDKERNEL_FOUNDATION_AND_SERVICES.md) §4.1 业务服务包标准结构中的"角色"维度，把培训材料按 **3 个核心角色** 拆分。

每份培训材料覆盖：

- 角色职责
- 7 步流中的具体动作
- 证据链中的留痕位置
- 异常路径与降级
