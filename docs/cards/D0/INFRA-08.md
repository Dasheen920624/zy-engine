## 目标

交付**会话安全闭环**：token 过期自动跳登录 + 多 tab storage event 同步（一处登出全部登出）+ 长时间无操作自动登出——防会话残留与多 tab 状态不一致，满足等保会话安全要求。

## 功能要求（原子可测条目）

- [ ] **FR-1 token 过期自动跳登录**：token/会话过期 → 自动跳登录（与 [INFRA-04](INFRA-04.md) 401 拦截同源）。
- [ ] **FR-2 多 tab 同步**：storage event 监听，一个 tab 登出 → 所有 tab 同步登出。
- [ ] **FR-3 无操作自动登出**：可配置无操作超时（核心 #19 配置外置，经 [CONFIG-01](CONFIG-01.md)）到期自动登出。
- [ ] **FR-4 会话续期**：活动时滑动续期（在最大会话时长内）。
- [ ] **FR-5 退出全 tab 同步**：主动退出在所有 tab 生效。
- [ ] **FR-6 超时前提醒**：无操作即将超时前提示（可续期/即将登出）。
