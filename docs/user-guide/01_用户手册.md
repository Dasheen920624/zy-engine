## 1. 路径引擎

### 1.1 概述

路径引擎用于管理临床路径模板，将诊疗规范转化为可执行的结构化路径，支持路径实例化、变异记录和统计分析。

### 1.2 功能清单

| 功能 | 菜单路径 | 说明 |
|------|----------|------|
| 路径模板列表 | 路径管理 → 路径模板 | 查看、创建、编辑、发布路径模板 |
| 路径模板详情 | 点击模板名称 | 查看路径节点、阶段、条件 |
| 路径实例列表 | 路径管理 → 患者路径 | 查看患者路径执行情况 |
| 变异记录 | 路径管理 → 变异记录 | 查看路径变异原因和统计 |

### 1.3 常用操作

#### 创建路径模板

1. 进入「路径管理 → 路径模板」页面
2. 点击「新建模板」按钮
3. 填写模板基本信息：名称、编码、适用科室、适用病种
4. 添加路径阶段（入院评估、诊疗执行、出院评估）
5. 在每个阶段添加诊疗项目
6. 点击「保存草稿」

#### 发布路径模板

1. 在路径模板列表中，选择状态为「草稿」的模板
2. 点击「发布」按钮
3. 系统自动校验模板完整性
4. 校验通过后，模板状态变为「已发布」

#### 查看患者路径

1. 进入「路径管理 → 患者路径」页面
2. 可按患者姓名、路径名称、状态筛选
3. 点击实例查看详细执行情况
4. 查看当前节点、已完成节点、变异记录

## 2. 规则引擎

### 2.1 概述

规则引擎用于管理临床决策规则，支持用药安全、医嘱合理性、医保合规等场景的自动检查和预警。

### 2.2 功能清单

| 功能 | 菜单路径 | 说明 |
|------|----------|------|
| 规则列表 | 规则管理 → 规则列表 | 查看、创建、编辑、发布规则 |
| 规则详情 | 点击规则名称 | 查看规则条件、动作、来源 |
| 规则验证 | 规则管理 → 规则验证 | 测试规则执行效果 |
| 规则包 | 规则管理 → 规则包 | 管理规则包的导入导出 |

### 2.3 常用操作

#### 创建规则

1. 进入「规则管理 → 规则列表」页面
2. 点击「新建规则」按钮
3. 填写规则基本信息：名称、编码、分类、优先级
4. 定义规则条件（WHEN）：选择触发条件
5. 定义规则动作（THEN）：选择执行动作
6. 点击「保存草稿」

#### 验证规则

1. 在规则列表中，选择需要验证的规则
2. 点击「验证」按钮
3. 输入测试患者上下文
4. 点击「执行验证」
5. 查看验证结果：命中规则、命中原因、建议操作

#### 发布规则

1. 在规则列表中，选择状态为「草稿」的规则
2. 点击「发布」按钮
3. 系统自动校验规则完整性
4. 校验通过后，规则状态变为「已发布」

## 3. 质控管理

### 3.1 概述

质控管理模块提供医疗质量评估、指标监控、安全红线检查等功能，支持病历质控、医保审核、AI 安全评估等场景。

### 3.2 功能清单

| 功能 | 菜单路径 | 说明 |
|------|----------|------|
| 评估指标集 | 质控管理 → 评估指标集 | 管理质控评估指标 |
| 评估报告 | 质控管理 → 评估报告 | 查看评估结果和报告 |
| AI 安全评估 | 质控管理 → AI 安全 | 红队测试、幻觉检测 |
| AI 治理 | 质控管理 → AI 治理 | 模型注册、Prompt 管理、评估任务 |

### 3.3 常用操作

#### 创建评估指标集

1. 进入「质控管理 → 评估指标集」页面
2. 点击「新建指标集」按钮
3. 填写指标集名称、编码、描述
4. 添加评估指标：名称、计算表达式、阈值、风险等级
5. 点击「保存草稿」

#### 执行评估

1. 在指标集列表中，选择已发布的指标集
2. 点击「执行评估」按钮
3. 选择评估对象和输入数据
4. 点击「开始评估」
5. 查看评估结果：命中指标、风险等级、建议措施

#### 查看评估报告

1. 进入「质控管理 → 评估报告」页面
2. 选择需要查看的报告
3. 查看评估摘要、命中指标详情、整改建议
4. 可导出报告（PDF/Excel）

## 4. 知识管理

### 4.1 概述

知识管理模块提供知识来源注册、订阅同步、AI 候选审核、知识图谱查询等功能，支持知识的全生命周期管理。

### 4.2 功能清单

| 功能 | 菜单路径 | 说明 |
|------|----------|------|
| 知识来源 | 知识管理 → 知识来源 | 注册和管理知识来源 |
| 知识订阅 | 知识管理 → 知识订阅 | 管理知识同步订阅 |
| AI 知识审核 | 知识管理 → AI 审核 | 审核 AI 生成的候选知识 |
| 知识图谱 | 知识管理 → 知识图谱 | 查询和浏览知识图谱 |
| 来源追溯 | 知识管理 → 来源追溯 | 追溯知识来源和引用 |

### 4.3 常用操作

#### 注册知识来源

1. 进入「知识管理 → 知识来源」页面
2. 点击「注册来源」按钮
3. 填写来源信息：名称、类型、连接配置
4. 点击「保存」

#### 审核 AI 候选知识

1. 进入「知识管理 → AI 审核」页面
2. 查看待审核的 AI 候选列表
3. 点击候选查看详情
4. 选择审核操作：
   - **通过**：确认知识正确，加入知识库
   - **拒绝**：知识有误，拒绝入库
   - **修改后通过**：修正内容后通过
5. 支持批量审核

#### 查询知识图谱

1. 进入「知识管理 → 知识图谱」页面
2. 输入查询条件（如疾病名称）
3. 点击「查询候选」
4. 查看图谱节点和关系
5. 展开「技术详情」查看节点/边/版本信息

## 附录

### A. 常见问题

**Q: 忘记密码怎么办？**
A: 请联系系统管理员重置密码。

**Q: 规则发布后可以修改吗？**
A: 已发布的规则不能直接修改，需要创建新版本。旧版本可标记为「已废弃」。

**Q: 如何导出数据？**
A: 在列表页面点击「导出」按钮，选择导出格式（Excel/CSV）。

**Q: 系统支持哪些浏览器？**
A: 推荐使用 Chrome 90+、Edge 90+、Firefox 88+。国产浏览器请使用极速模式。
