> 版本：1.0 | 适用：v1.0 GA | 日期：2026-05-23
> 交付对象：医院信息科 / 临床科室 / 医务人员

## 一、临床路径管理模块

### 1.1 模块概述

临床路径管理模块提供临床路径的定义、版本管理、患者实例化和变异记录功能，帮助医院标准化诊疗流程，提升医疗质量。

### 1.2 功能清单

| 功能 | 说明 | 操作角色 |
|------|------|---------|
| 路径定义 | 创建和管理临床路径模板 | 路径管理员 |
| 版本管理 | 路径版本发布、审核、停用 | 路径管理员 |
| 患者入径 | 将患者纳入临床路径 | 临床医生 |
| 节点执行 | 按路径节点推进诊疗流程 | 临床医生 |
| 变异记录 | 记录偏离路径的变异情况 | 临床医生 |
| 推荐记录 | AI 辅助推荐记录 | 临床医生 |

### 1.3 操作指南

#### 1.3.1 创建路径定义

1. 进入 **临床路径 > 路径定义** 页面
2. 点击 **新建路径** 按钮
3. 填写路径基本信息：
   - 路径名称（必填）
   - 适用科室（必填）
   - 适用病种（必填）
   - 路径描述
4. 添加路径节点：
   - 点击 **添加节点**
   - 设置节点名称、类型（评估/检查/治疗/出院）
   - 设置节点顺序和条件
5. 点击 **保存** 完成创建

#### 1.3.2 患者入径

1. 进入 **临床路径 > 患者路径** 页面
2. 点击 **患者入径** 按钮
3. 选择患者和适用的路径定义
4. 确认入径信息
5. 系统自动创建患者路径实例

#### 1.3.3 记录变异

1. 在患者路径详情页，点击当前节点
2. 点击 **记录变异** 按钮
3. 选择变异类型（必须变异 / 可选变异 / 系统变异）
4. 填写变异原因和说明
5. 点击 **保存** 完成记录

### 1.4 常见问题

**Q: 路径定义发布后能否修改？**
A: 已发布的路径定义不可直接修改，需创建新版本。进入路径定义详情，点击"创建新版本"。

**Q: 患者入径后如何退出路径？**
A: 在患者路径详情页，点击"退出路径"，选择退出原因即可。

**Q: 变异率如何查看？**
A: 在路径定义详情页的"统计"标签下，可查看该路径的变异率统计。

## 二、规则引擎模块

### 2.1 模块概述

规则引擎模块提供临床规则的创建、管理和执行功能，支持基于患者数据的自动预警和推荐，辅助临床决策。

### 2.2 功能清单

| 功能 | 说明 | 操作角色 |
|------|------|---------|
| 规则定义 | 创建和管理临床规则 | 规则管理员 |
| 规则分组 | 按科室/病种分组管理 | 规则管理员 |
| 规则执行 | 自动/手动触发规则 | 系统/临床医生 |
| 执行日志 | 查看规则执行记录 | 规则管理员 |
| 规则测试 | 模拟测试规则效果 | 规则管理员 |

### 2.3 操作指南

#### 2.3.1 创建规则

1. 进入 **规则引擎 > 规则定义** 页面
2. 点击 **新建规则** 按钮
3. 填写规则基本信息：
   - 规则名称（必填）
   - 规则编码（必填，唯一标识）
   - 适用科室
   - 优先级
4. 配置触发条件：
   - 选择触发事件类型
   - 设置条件表达式
5. 配置执行动作：
   - 选择动作类型（预警/推荐/拦截）
   - 设置动作参数
6. 点击 **保存** 完成创建

#### 2.3.2 测试规则

1. 在规则定义详情页，点击 **测试** 按钮
2. 输入模拟患者数据
3. 点击 **执行测试**
4. 查看规则执行结果和触发的动作

#### 2.3.3 查看执行日志

1. 进入 **规则引擎 > 执行日志** 页面
2. 可按时间范围、规则名称、执行结果筛选
3. 点击日志条目查看详细执行过程

### 2.4 常见问题

**Q: 规则执行失败如何排查？**
A: 在执行日志中查看失败原因。常见原因包括：条件表达式语法错误、引用的数据字段不存在、动作参数配置不完整。

**Q: 规则优先级如何生效？**
A: 当多条规则同时触发时，按优先级从高到低执行。同优先级按创建时间排序。

## 三、术语服务模块

### 3.1 模块概述

术语服务模块提供标准化医学术语的管理和映射功能，支持 ICD-10、SNOMED CT 等标准术语体系，实现不同系统间的术语互操作。

### 3.2 功能清单

| 功能 | 说明 | 操作角色 |
|------|------|---------|
| 标准概念 | 管理标准术语概念 | 术语管理员 |
| 概念映射 | 建立不同术语体系间的映射 | 术语管理员 |
| 未映射队列 | 处理无法自动映射的术语 | 术语管理员 |
| 概念搜索 | 检索标准术语 | 所有用户 |

### 3.3 操作指南

#### 3.3.1 查看标准概念

1. 进入 **术语服务 > 标准概念** 页面
2. 使用搜索框按编码或名称检索
3. 点击概念条目查看详情（编码、名称、定义、所属体系）

#### 3.3.2 创建概念映射

1. 进入 **术语服务 > 概念映射** 页面
2. 点击 **新建映射** 按钮
3. 选择源概念和目标概念
4. 设置映射类型（等价/宽泛/狭窄/部分）
5. 填写映射依据
6. 点击 **保存** 完成创建

#### 3.3.3 处理未映射队列

1. 进入 **术语服务 > 未映射队列** 页面
2. 查看待处理的未映射术语
3. 选择处理方式：
   - 映射到已有标准概念
   - 创建新的标准概念
   - 标记为暂不处理
4. 点击 **提交** 完成处理

### 3.4 常见问题

**Q: 如何批量导入术语映射？**
A: 目前暂不支持批量导入，需逐条创建。后续版本将支持 CSV 批量导入。

**Q: 术语映射审核流程是什么？**
A: 新建映射需经术语管理员审核通过后生效。审核周期一般为 3 个工作日。

## 四、知识图谱模块

### 4.1 模块概述

知识图谱模块提供医学知识图谱的可视化浏览和查询功能，支持疾病-症状-药物-检查等实体间的关联关系展示，辅助临床知识发现。

### 4.2 功能清单

| 功能 | 说明 | 操作角色 |
|------|------|---------|
| 图谱浏览 | 可视化浏览知识图谱 | 所有用户 |
| 实体搜索 | 按名称/编码搜索实体 | 所有用户 |
| 关系查询 | 查询实体间的关系 | 所有用户 |
| 图谱版本 | 管理图谱版本 | 图谱管理员 |
| 来源追溯 | 追溯知识来源和证据 | 所有用户 |

### 4.3 操作指南

#### 4.3.1 浏览知识图谱

1. 进入 **知识图谱 > 图谱浏览** 页面
2. 在搜索框中输入实体名称（如疾病名称）
3. 点击搜索结果中的实体，图谱将以该实体为中心展开
4. 使用鼠标拖拽和滚轮缩放浏览图谱
5. 点击实体节点查看详细信息
6. 点击关系边查看关系类型和来源

#### 4.3.2 查询实体关系

1. 进入 **知识图谱 > 关系查询** 页面
2. 输入起始实体和目标实体
3. 选择关系类型（可选）
4. 点击 **查询** 按钮
5. 查看查询结果：关系路径、关系类型、证据来源

#### 4.3.3 查看来源追溯

1. 在图谱浏览或关系查询结果中，点击任意知识条目
2. 点击 **来源追溯** 标签
3. 查看知识的来源文献、指南引用、证据等级

### 4.4 常见问题

**Q: 图谱数据来源是什么？**
A: 知识图谱数据来源于权威医学文献、临床指南和专家知识，经过专业团队审核后入库。

**Q: 图谱数据更新频率？**
A: 知识图谱按季度更新，重大临床指南变更时即时更新。

**Q: 图谱不可用时怎么办？**
A: 知识图谱为可选 Provider，不可用时系统自动降级，不影响核心路径和规则功能。
