# MedKernel 等保 2.0 三级控制点自查清单

> 版本：1.0 | 适用：v1.0 GA | 日期：2026-05-23
> 依据：GB/T 22239-2019《信息安全技术 网络安全等级保护基本要求》第三级

## 1. 概述

本文档依据等保 2.0 三级要求，对 MedKernel v1.0 GA 进行逐项自查，
明确合规状态、证据和整改计划，为医院客户等保测评提供基础材料。

**合规状态定义**：
- ✅ 合规：已实现，有证据
- ⚠️ 部分合规：已实现部分，需补充
- ❌ 不合规：未实现，需整改
- ➖ 不适用：该控制点不适用于本系统
