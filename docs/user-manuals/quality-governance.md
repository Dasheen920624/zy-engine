# 质控治理模块用户手册

## 1. 模块概述

质控治理模块帮助医院管理医疗质量控制。通过本模块，您可以：

- 通过质控驾驶舱实时掌握全院质量指标运行情况
- 管理质控预警，及时发现和处理质量问题
- 配置和管理评估指标集，建立科学的质控评估体系
- 执行质控评估并生成评估报告
- 按科室、时间等维度进行深度分析和趋势追踪

> **适用人员**：医院信息科管理人员、质控管理人员

---

## 2. 快速入门

### 2.1 查看质控驾驶舱

1. 在左侧菜单中点击「质控治理」
2. 默认进入质控驾驶舱页面

[截图：质控驾驶舱页面]

驾驶舱页面展示以下核心内容：

- **KPI 指标卡片**：关键质量指标的实时数值和同比环比变化
- **科室排名**：各科室质量指标的综合排名
- **趋势图**：关键指标的时间趋势变化
- **预警概览**：当前未处理预警的数量和等级分布

### 2.2 查看预警列表

1. 在驾驶舱页面点击预警概览区域的「查看全部」
2. 或在左侧菜单中点击「预警管理」
3. 进入预警列表页面

[截图：预警列表页面]

预警列表显示以下信息：

| 列名 | 说明 |
|------|------|
| 预警编号 | 系统自动生成的唯一编号 |
| 预警等级 | 紧急 / 重要 / 一般 |
| 预警类型 | 指标异常 / 流程违规 / 数据缺失等 |
| 预警内容 | 预警的具体描述 |
| 状态 | 待处理 / 处理中 / 已关闭 |
| 生成时间 | 预警产生的时间 |

## 3. 核心操作

### 3.1 质控驾驶舱

质控驾驶舱是质控治理的核心入口，提供全院质量状况的全景视图。

#### 查看 KPI 指标

1. 在驾驶舱页面顶部，展示关键 KPI 指标卡片
2. 每个卡片显示：
   - **当前值**：指标的最新数值
   - **目标值**：指标的达标标准
   - **同比变化**：与去年同期相比的变化
   - **环比变化**：与上月相比的变化
   - **达标状态**：达标（绿色）/ 预警（黄色）/ 不达标（红色）

[截图：KPI 指标卡片区域]

#### 查看科室排名

1. 在驾驶舱页面中部，展示科室质量排名
2. 排名依据各科室的综合质量得分
3. 点击科室名称可进入科室详情页面

[截图：科室排名区域]

#### 查看趋势图

1. 在驾驶舱页面下方，展示关键指标的趋势图
2. 可切换查看不同指标的趋势
3. 支持选择时间范围（近 7 天 / 近 30 天 / 近 90 天 / 自定义）

[截图：趋势图区域]

### 3.2 预警管理

#### 查看预警

1. 进入预警列表页面
2. 可通过筛选条件按预警等级、类型、状态、时间范围过滤
3. 点击预警编号可查看预警详情

[截图：预警详情页面]

预警详情页面包含：

- **预警信息**：等级、类型、内容、生成时间
- **关联指标**：触发该预警的指标及数值
- **关联患者**：涉及的患者信息（如适用）
- **处理记录**：该预警的所有处理记录

#### 分配处理人

1. 在预警列表或详情页面，点击「分配」按钮
2. 选择处理人（可按科室、角色筛选）
3. 填写分配说明（可选）
4. 点击「确认分配」

[截图：预警分配对话框]

> **提示**：预警分配后，处理人会收到系统通知。

#### 处理预警

1. 被分配的预警会出现在处理人的「我的预警」列表中
2. 点击预警进入详情页面
3. 点击「开始处理」，预警状态变为「处理中」
4. 填写处理措施和处理结果
5. 点击「提交处理」

[截图：预警处理页面]

#### 关闭预警

1. 预警处理完成后，点击「关闭预警」
2. 填写关闭说明
3. 点击「确认关闭」，预警状态变为「已关闭」

> **注意**：只有预警的创建人或管理员可以关闭预警。

### 3.3 评估指标集

评估指标集是质控评估的基础，包含一组相关的质量指标。

#### 查看指标

1. 在左侧菜单中点击「评估指标集」
2. 进入指标集列表页面
3. 点击指标集名称可查看其中包含的所有指标

[截图：指标集列表页面]

每个指标显示以下信息：

| 列名 | 说明 |
|------|------|
| 指标名称 | 指标的显示名称 |
| 指标编码 | 系统自动生成的唯一编号 |
| 数据来源 | 指标数据的来源系统 |
| 计算方式 | 指标的计算公式或规则 |
| 目标值 | 指标的达标标准 |
| 单位 | 指标的计量单位 |

#### 新建指标集

1. 在指标集列表页面，点击「新建指标集」按钮
2. 填写基本信息：
   - **指标集名称**：如"住院医疗质量指标集"
   - **适用范围**：全院 / 指定科室
   - **描述**：指标集的用途说明
3. 点击「保存」

[截图：新建指标集页面]

#### 配置指标

1. 在指标集详情页面，点击「添加指标」按钮
2. 可从已有指标库中选择指标，或新建自定义指标
3. 新建自定义指标需填写：
   - **指标名称**和**指标编码**
   - **数据来源**：选择数据取值来源
   - **计算方式**：设置指标的计算公式
   - **目标值**：设置达标标准
   - **预警阈值**：设置触发预警的阈值
4. 配置完成后点击「保存」

[截图：指标配置页面]

### 3.4 执行评估

#### 选择指标集

1. 在左侧菜单中点击「执行评估」
2. 选择要使用的指标集
3. 选择评估范围：
   - 全院评估
   - 按科室评估
   - 按病种评估

[截图：评估执行页面]

#### 执行评估

1. 确认评估参数后，点击「开始执行」
2. 系统自动采集数据并计算各项指标
3. 执行过程中页面显示进度条
4. 执行时间取决于数据量和指标复杂度

#### 查看结果

1. 评估完成后，系统自动跳转到结果页面
2. 结果页面展示：
   - **综合得分**：整体质量评估得分
   - **各指标得分**：每个指标的达标情况和得分
   - **不达标项**：未达标的指标列表及差距分析
   - **同比环比**：与历史评估结果的对比

[截图：评估结果页面]

### 3.5 评估报告

#### 生成报告

1. 在评估结果页面，点击「生成报告」按钮
2. 选择报告模板：
   - 标准报告：包含所有指标的完整分析
   - 简要报告：仅包含关键指标和结论
   - 专项报告：针对特定指标的深度分析
3. 点击「生成」

[截图：报告生成页面]

#### 导出 PDF

1. 报告生成后，在报告预览页面点击「导出 PDF」按钮
2. 选择导出选项：
   - 包含图表：报告中嵌入统计图表
   - 包含明细：报告中包含指标计算明细
   - 水印设置：添加"内部使用"等水印
3. 点击「确认导出」，PDF 文件自动下载到本地

[截图：报告导出选项页面]

## 4. 高级功能

### 4.1 科室钻取

科室钻取功能帮助您从全院视角逐层深入到科室、医生层面。

1. 在质控驾驶舱或评估结果页面，点击科室名称
2. 进入科室详情页面，展示该科室的详细指标

[截图：科室钻取页面]

科室详情页面包含：

- **科室概览**：科室基本情况和综合得分
- **指标详情**：科室各质量指标的详细数值和趋势
- **医生绩效**：科室各医生的质量指标完成情况

#### 查看医生绩效

1. 在科室详情页面，点击「医生绩效」标签
2. 列表展示科室所有医生的指标完成情况
3. 可按指标排序，快速识别表现突出或需改进的医生

[截图：医生绩效页面]

### 4.2 趋势分析

趋势分析功能帮助您追踪质量指标的变化趋势，发现潜在问题。

1. 在左侧菜单中点击「趋势分析」
2. 选择分析参数：
   - **指标选择**：选择要分析的指标（可多选）
   - **时间范围**：选择分析的时间段
   - **对比维度**：同比 / 环比 / 科室对比
3. 点击「生成分析」

[截图：趋势分析页面]

分析结果展示：

- **趋势折线图**：指标随时间的变化曲线
- **关键拐点标注**：指标发生显著变化的时点
- **原因关联**：与指标变化可能相关的因素提示
- **预测趋势**：基于历史数据的短期趋势预测

## 5. 常见问题

### Q1：评估结果不准确怎么办？

评估结果不准确可能由以下原因导致：

1. **数据源问题**：源头系统的数据未及时更新，请确认数据同步状态
2. **指标配置错误**：检查指标的计算方式和数据来源配置是否正确
3. **时间范围不当**：评估的时间范围过短可能导致数据量不足，建议扩大时间范围
4. **数据缺失**：部分数据未采集到，请检查数据采集任务是否正常运行

建议重新执行评估，如果问题仍然存在，请联系系统管理员检查数据源配置。

### Q2：预警如何批量处理？

1. 在预警列表页面，勾选需要批量处理的预警
2. 点击列表上方的「批量处理」按钮
3. 选择批量操作类型：
   - **批量分配**：将选中的预警统一分配给同一处理人
   - **批量关闭**：将选中的预警统一关闭（需填写关闭原因）
4. 确认操作

> **注意**：批量关闭操作需谨慎，建议仅对同一类型的预警进行批量关闭。

### Q3：报告导出失败怎么办？

报告导出失败通常有以下原因：

1. **报告数据量过大**：包含明细数据的大型报告可能导出超时，建议取消"包含明细"选项后重试
2. **浏览器限制**：部分浏览器可能阻止文件下载，请检查浏览器下载设置
3. **网络中断**：导出过程中网络不稳定可能导致失败，请检查网络连接后重试
4. **系统资源不足**：服务器资源紧张时可能影响导出，请稍后重试

如果反复导出失败，请联系系统管理员。
