# 路径治理模块用户手册

## 1. 模块概述

路径治理模块帮助医院管理临床路径。通过本模块，您可以：

- 设计和管理标准化的临床路径，规范诊疗流程
- 对临床路径进行新建、编辑、发布和版本管理
- 管理患者入径、出径和变异记录
- 通过可视化编辑器直观地设计路径流程
- 统计分析路径执行情况，持续优化路径质量

> **适用人员**：医院信息科管理人员、临床路径管理员

---

## 2. 快速入门

### 2.1 查看路径列表

1. 在左侧菜单中点击「路径治理」
2. 进入路径列表页面，默认显示所有已发布的路径

[截图：路径列表页面]

列表中每条路径显示以下信息：

| 列名 | 说明 |
|------|------|
| 路径名称 | 路径的显示名称 |
| 路径编码 | 系统自动生成的唯一编号 |
| 适用病种 | 路径适用的疾病诊断 |
| 状态 | 草稿 / 已发布 / 已停用 |
| 当前版本 | 路径的当前版本号 |
| 更新时间 | 最近一次修改的时间 |

### 2.2 搜索路径

1. 在路径列表页面上方的搜索框中输入关键词
2. 支持按路径名称、路径编码、适用病种搜索
3. 可使用筛选条件按状态、科室、病种类型进行过滤

[截图：搜索与筛选区域]

### 2.3 查看路径详情

1. 在路径列表中点击路径名称
2. 进入路径详情页面，可查看路径的完整配置信息

[截图：路径详情页面]

详情页面包含以下标签页：

- **基本信息**：路径名称、编码、适用病种、预计住院天数等
- **阶段设计**：路径的各阶段划分及时间安排
- **节点配置**：每个阶段内的诊疗节点详情
- **版本历史**：路径的所有历史版本记录

## 3. 核心操作

### 3.1 新建路径

#### 步骤一：填写基本信息

1. 点击路径列表页面的「新建路径」按钮
2. 填写路径基本信息：
   - **路径名称**：如"急性心肌梗死介入治疗路径"
   - **适用病种**：选择关联的 ICD 诊断编码
   - **适用科室**：选择执行该路径的科室
   - **预计住院天数**：标准住院天数
   - **路径描述**：简要说明路径的适用范围和目标

[截图：路径基本信息填写页面]

#### 步骤二：设计阶段

1. 切换到「阶段设计」标签
2. 点击「添加阶段」按钮
3. 为每个阶段设置：
   - **阶段名称**：如"入院评估期"、"治疗执行期"、"出院准备期"
   - **时间范围**：该阶段的起止天数（如第 1-2 天）
   - **阶段目标**：该阶段需要完成的诊疗目标
4. 通过拖拽调整阶段的顺序

[截图：阶段设计页面]

#### 步骤三：配置节点

1. 切换到「节点配置」标签
2. 进入路径编辑器画布
3. 在对应阶段内添加诊疗节点：
   - **医嘱节点**：标准用药、检查检验等医嘱
   - **评估节点**：病情评估、风险评分等
   - **操作节点**：手术、操作等
   - **判断节点**：条件分支，根据评估结果选择不同路径
4. 设置节点之间的连线关系，定义执行顺序

[截图：节点配置画布页面]

#### 步骤四：保存草稿

1. 完成阶段和节点配置后，点击「保存草稿」按钮
2. 系统会自动校验路径配置的完整性和逻辑正确性
3. 校验通过后，路径状态变为「草稿」

> **注意**：草稿状态的路径不能用于患者入径。

#### 步骤五：发布路径

1. 确认路径配置无误后，点击「发布」按钮
2. 系统会进行完整校验，确认路径可以正常使用
3. 校验通过后，路径状态变为「已发布」，版本号自动递增

[截图：路径发布确认对话框]

### 3.2 编辑路径

1. 在路径列表中找到需要编辑的路径，点击「编辑」按钮
2. 进入路径编辑页面，修改阶段或节点配置
3. 修改完成后，点击「保存草稿」保存修改
4. 确认无误后，点击「发布」使修改生效

> **说明**：编辑已发布的路径时，系统会创建新版本，原版本不受影响。正在使用旧版本的患者继续按旧版本执行。

### 3.3 路径对比

路径对比功能帮助您查看不同版本之间的差异。

1. 在路径详情页面，点击「版本对比」按钮
2. 选择需要对比的两个版本
3. 系统会以并排方式展示两个版本的差异：
   - **新增内容**：以绿色标记
   - **删除内容**：以红色标记
   - **修改内容**：以黄色标记

[截图：路径版本对比页面]

对比结果包括：

- 基本信息的变更
- 阶段增减和调整
- 节点的增删改
- 连线关系的变更

### 3.4 患者入径

#### 步骤一：选择患者

1. 在左侧菜单中点击「患者入径」
2. 在患者列表中搜索或筛选目标患者
3. 点击患者行的「入径」按钮

[截图：患者入径选择页面]

#### 步骤二：选择路径

1. 系统根据患者诊断自动推荐匹配的临床路径
2. 您也可以手动搜索和选择其他路径
3. 确认选择的路径后，点击「下一步」

#### 步骤三：确认入径

1. 系统显示路径概要信息，包括预计住院天数、主要诊疗节点等
2. 确认信息无误后，点击「确认入径」
3. 患者正式进入该临床路径，开始按路径执行诊疗流程

[截图：入径确认页面]

> **提示**：入径后，患者的诊疗流程将按照路径节点逐步推进，医生可在路径执行页面查看当前进度。

### 3.5 路径变异管理

变异是指患者在路径执行过程中偏离标准流程的情况。

#### 记录变异

1. 在患者路径执行页面，点击「记录变异」按钮
2. 填写变异信息：
   - **变异类型**：患者原因 / 医护原因 / 系统原因
   - **变异描述**：详细说明变异情况
   - **处理措施**：针对变异采取的处理方式
3. 点击「保存」

[截图：变异记录填写页面]

#### 分析变异原因

1. 在左侧菜单中点击「变异分析」
2. 选择分析维度（按路径、按科室、按时间段等）
3. 系统生成变异分析报告，包括：
   - 变异发生率统计
   - 变异类型分布
   - 高频变异原因排名
   - 变异趋势变化

[截图：变异分析报告页面]

## 4. 高级功能

### 4.1 路径编辑器

路径编辑器提供可视化的画布操作界面，方便您直观地设计路径流程。

#### 画布操作

| 操作 | 方法 |
|------|------|
| 缩放画布 | 鼠标滚轮 / 工具栏缩放按钮 |
| 平移画布 | 按住鼠标右键拖拽 / 按住空格键拖拽 |
| 适应窗口 | 点击工具栏「适应窗口」按钮 |
| 撤销/重做 | Ctrl+Z / Ctrl+Y |

#### 拖拽节点

1. 从左侧节点面板中拖拽节点类型到画布上
2. 节点放置后，可继续拖拽调整位置
3. 双击节点可编辑节点详细配置

#### 连线操作

1. 将鼠标移到节点边缘的连接点上
2. 按住鼠标左键拖拽到目标节点的连接点
3. 释放鼠标完成连线
4. 对于判断节点，可为不同分支设置条件标签

[截图：路径编辑器画布操作示意]

### 4.2 路径统计

路径统计页面提供路径执行情况的整体分析。

1. 在左侧菜单中点击「路径统计」
2. 进入统计概览页面

[截图：路径统计概览页面]

#### 核心指标

| 指标 | 说明 |
|------|------|
| 入径率 | 入径患者数占符合入径条件患者数的比例 |
| 完成率 | 完成路径的患者数占入径患者数的比例 |
| 变异率 | 发生变异的患者数占入径患者数的比例 |
| 平均住院日 | 入径患者的平均实际住院天数 |
| 路径执行率 | 实际执行节点数占应执行节点数的比例 |

#### 统计维度

- **按路径**：查看各路径的执行情况
- **按科室**：查看各科室的路径执行情况
- **按时间段**：查看不同时间段的趋势变化
- **按医生**：查看各医生的路径执行情况

## 5. 常见问题

### Q1：路径发布失败怎么办？

路径发布失败通常有以下原因：

1. **必填信息不完整**：请检查路径名称、适用病种等必填项是否已填写
2. **阶段未配置节点**：每个阶段至少需要配置一个节点
3. **节点连线不完整**：判断节点的所有分支都需要有连线
4. **时间范围冲突**：阶段之间的时间范围不能重叠
5. **存在孤立节点**：画布中不能有未连接的节点

请根据系统提示的错误信息逐一修正后重新发布。

### Q2：患者如何退出路径？

1. 在患者路径执行页面，点击「退出路径」按钮
2. 选择退出原因：
   - 患者转院
   - 病情变化不再适用
   - 患者拒绝继续
   - 其他原因
3. 填写退出说明
4. 点击「确认退出」

> **注意**：退出路径后，该患者之前的路径执行记录仍会保留，可在历史记录中查看。

### Q3：变异记录怎么查看？

1. 在患者路径执行页面，点击「变异记录」标签
2. 可查看该患者的所有变异记录列表
3. 点击单条记录可查看详细信息

如需查看全院变异情况，请到「变异分析」页面，通过筛选条件查看不同维度的变异数据。
